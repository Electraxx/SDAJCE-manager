Sommaire

description

ce programme vous permet de lister les différents cartes présente dans le jeu de carte évolutif - seigneur des anneaux, d'ajouter de nouvelles cartes, de créer et interagir avec les différents decks que vous avez créé et de voir les parties que vous avez jouée

Descriptif des commandes

  • ctrl+q : vous permet de quitter l'application
  • ctrl+f : vous permet d'accéder à la page d'ajout d'une carte
  • ctrl+d : vous permet d'accéder à la page de création d'un deck
  • ctrl+p : vous permet d'accéder à la page de création d'une partie

lorsque vous êtes sur une page d'ajout quelque soit celle où vous vous trouvez, si vous ne voulez pas ajouter quelque chose, cliquez simplement sur les des onglets de la toolBar
